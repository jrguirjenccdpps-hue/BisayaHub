&gt; **Learn Bisaya (Cebuano) for free. Open source. Forever.**

[BisayaHub](https://yourusername.github.io/bisayahub) is a free, interactive web application for learning Bisaya language - spoken by 20+ million Filipinos in Visayas and Mindanao.

## ✨ Features

- 💬 **Conversational AI** - Chat naturally, get instant translations
- 📚 **Structured Learning** - Modules for greetings, food, travel, etc.
- 🎯 **Practice Games** - Quizzes, flashcards, matching games
- 📊 **Progress Tracking** - XP, levels, streaks, achievements
- 🧠 **Smart Analysis** - Understands sentences, not just words
- 🌍 **Open Source** - Free forever, community-driven

## 🚀 Quick Start

### For Learners
